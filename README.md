# ShopGen

ShopGen — это веб-приложение для автоматической генерации описаний товаров для интернет-магазина.

Пользователь выбирает категорию товара, заполняет характеристики, нажимает кнопку генерации и получает готовое текстовое описание. Проект помогает быстрее создавать карточки товаров, сохранять единый стиль описаний и уменьшать объем ручной работы.

## Демо

Сайт: https://c-site-project.onrender.com/

## Основная идея

В интернет-магазинах часто нужно создавать много похожих карточек товаров. Ручное написание описаний занимает много времени, особенно если каталог постоянно пополняется.

ShopGen автоматизирует этот процесс. Вместо ручного написания текста сервис использует шаблоны, словари фраз и параметры товара. На основе введенных данных формируется связное описание, которое можно использовать как готовый текст или основу для дальнейшей правки.

## Что умеет проект

- генерировать описание товара по характеристикам;
- поддерживать несколько категорий товаров;
- менять поля формы в зависимости от выбранной категории;
- использовать шаблонную генерацию текста;
- добавлять вариативность через словари фраз;
- использовать seed для воспроизводимого результата;
- сохранять историю генераций;
- фильтровать историю по категориям;
- работать через веб-интерфейс;
- работать через REST API;
- обрабатывать один товар или список товаров.

## Поддерживаемые категории

Сейчас реализованы 3 категории:

- смартфоны;
- кроссовки;
- видеокарты.

Для каждой категории используется свой набор характеристик и свои шаблоны описаний.
